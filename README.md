# Sales Analytics Dashboard using Excel

## Project Overview

This project is a comprehensive sales analytics dashboard built in Microsoft Excel to analyze business performance across revenue, profit, regions, countries, product categories, and sales channels.

The objective of this project is to transform raw sales data into meaningful business insights using Excel’s data analysis and visualization capabilities.

---

## Tools & Technologies Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning
* Dashboard Design
* Business Analysis

---

## Key Business Objectives

* Analyze revenue and profit trends over time
* Identify top-performing and underperforming sales regions
* Determine best-selling countries and product categories
* Compare online vs offline sales performance
* Generate actionable business recommendations

---

### Channel Analysis

![Channel Analysis](assets/Channel%20Analysis.png)

---

## Key Insights

* Focus on underperforming regions to expand market reach
* Increase investment in high-margin product categories
* Optimize operational costs to improve profitability
* Strengthen festive/discount campaigns for online growth
* Improve offline distribution via retail partnerships
